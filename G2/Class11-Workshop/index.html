Exercise level 1

Create a javascript function which will accept an array of numbers as input and insert dashes (-) between each two even numbers. Make sure to implement validation of the input array.

Example: if you accept [0,2,5,4,6,8] the output should be [0,-,2,5,4,-,6,-,8].

Exercise level 2

Create an ATM machine. Use objects. The ATM has property balance. Initialize it with some value.

  • The ATM should have the following functionalities:
  • - make deposit (this function should add money to the balance)
  • - make withdrawal (this function should withdraw money)
  • - get the balance (this function should display the current balance)
  • -display interactive menu using prompt (1 -> get balance, 2-> make deposit, 3 -> withdrawal, 4 - exit).

Bonus: Use dom elements and events (ex. select, input, buttons..)

Exercise level 3

Create an interactive web app for displaying Star Wars people and ships.The app should look like the given image.

When you click on the cosmonaut image, the people should be displayed. Implement next and previous button functionalities. Use the following url https://swapi.dev/api/people/?page=1.

When you click on the rocket image, the ships should be displayed. Implement next and previous button functionalities. Use the following url https://swapi.dev/api/starships/?page=1.

Hint: for getting the previous and next page of data, you can use the properties from the ajax result, instead of manually updating page value.

If you are having COSR problem with swapi first try adding 

mode: "no-cors",
headers: {
        "Access-Control-Allow-Origin": "*",
      }
when making the ajax call. Else try pokeapi and implement click on the cosmonaut image. https://pokeapi.co/api/v2/pokemon?offset=0&limit=10. Here 0 means skip 0 results and return the first 10, if offset is 10 skip 10 results and return the next 10 etc. manipulate the offset or use the next and previous properties in the result to get the links that should be hit on next and previous buttons.
